## Schema Null

Tests that the not null of types are preserved

---

### Simple

**TypeDefs**

```typedefs-input
type Movie {
    id: ID!
    ids: [ID!]!
    name: String!
    names: [String!]!
    actorCount: Int!
    actorCounts: [Int!]!
    averageRating: Float!
    averageRatings: [Float!]!
    isActives: [Boolean!]!
    filmedAt: Point!
    filmedAts: [Point!]!
    createdAt: DateTime!
    createdAts: [DateTime!]!
}
```
