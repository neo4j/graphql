# Schema Unions

Tests that the provided typeDefs return the correct schema.

---

## Unions

### TypeDefs

```graphql
union Search = Movie | Genre

type Genre {
    id: ID
}

type Movie {
    id: ID
    search: [Search] @relationship(type: "SEARCH", direction: OUT)
    searchNoDirective: Search
}
```
